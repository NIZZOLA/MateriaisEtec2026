1. 🔁 Revisão rápida (5 min)

  • Variáveis, prompt, console.log, if/else
  • Problema: prompt é feio e limitado. Queremos interagir com a página de verdade!
  • Solução: JavaScript vai manipular formulários HTML.

2. 🌳 Introdução ao DOM (10 min)

DOM = Document Object Model
É uma representação dos elementos HTML que o JavaScript consegue enxergar e modificar.

Estrutura básica de um formulário HTML:

<input type="text" id="nome" placeholder="Digite seu nome">
<button id="btnEnviar">Enviar</button>
<div id="resultado"></div>

3. 🎣 Selecionando elementos e lendo valores (20 min)

3.1 Selecionar por id

let campoNome = document.getElementById("nome");
let botao = document.getElementById("btnEnviar");
let divResultado = document.getElementById("resultado");

3.2 Ler valor de um input

let valorDigitado = campoNome.value;
console.log(valorDigitado);

3.3 Escrever dentro de um elemento HTML

divResultado.innerHTML = "Olá, " + valorDigitado;

3.4 Evento de clique (onclick)

botao.onclick = function() {
    let nome = campoNome.value;
    divResultado.innerHTML = `Olá ${nome}! Bem-vindo(a).`;
};

🧪 Exemplo completo:

<!DOCTYPE html>
<html>
<head>
    <title>Exemplo DOM</title>
</head>
<body>
    <h2>Digite seu nome:</h2>
    <input type="text" id="nome">
    <button id="botao">Clique</button>
    <div id="mensagem"></div>

    <script>
        let inputNome = document.getElementById("nome");
        let botao = document.getElementById("botao");
        let divMensagem = document.getElementById("mensagem");

        botao.onclick = function() {
            let nome = inputNome.value;
            divMensagem.innerHTML = `<strong>Olá ${nome}!</strong>`;
        };
    </script>
</body>
</html>

4. 🧮 Exercício guiado 1 – Calculadora Simples (20 min)

Objetivo: Criar uma calculadora que soma dois números e exibe o resultado em um div.

📝 Estrutura HTML:

<input type="number" id="num1" placeholder="Primeiro número">
<input type="number" id="num2" placeholder="Segundo número">
<button id="somar">Somar</button>
<div id="resultado"></div>

💻 Código JavaScript:

let n1 = document.getElementById("num1");
let n2 = document.getElementById("num2");
let botaoSomar = document.getElementById("somar");
let divResultado = document.getElementById("resultado");

botaoSomar.onclick = function() {
    let valor1 = Number(n1.value);
    let valor2 = Number(n2.value);
    let soma = valor1 + valor2;
    
    divResultado.innerHTML = `A soma é: ${soma}`;
};
🔥 Desafio rápido: Adicionar botões para subtração, multiplicação e divisão.

5. ⚡ Eventos adicionais e escrita em componentes (15 min)

5.1 Evento oninput (enquanto digita)

n1.oninput = function() {
    console.log("Valor mudou para:", n1.value);
};

5.2 Escrever em outros elementos

  • span: texto inline
  • p: parágrafo
  • h3: título pequeno

5.3 Limpar campos após operação

n1.value = "";   // limpa o campo
n2.value = "";

5.4 Validar se campos estão vazios

if (n1.value === "" || n2.value === "") {
    alert("Preencha os dois números!");
    return;   // para a execução
}

6. ✅ Exercício prático – Validador de Formulário (20 min)

Enunciado: Criar um formulário de cadastro simples com validação JavaScript.

Campos:

  • Nome (obrigatório, mínimo 3 caracteres)
  • E-mail (obrigatório, deve conter "@")
  • Idade (obrigatório, entre 1 e 120)

Comportamento:

  • Ao clicar em "Validar", exibir mensagens de erro específicas
  • Se tudo estiver correto, mostrar "Cadastro válido!" em verde

🔧 Estrutura HTML sugerida:

<label>Nome:</label>
<input type="text" id="nome">
<span id="erroNome" style="color:red"></span><br>

<label>E-mail:</label>
<input type="email" id="email">
<span id="erroEmail" style="color:red"></span><br>

<label>Idade:</label>
<input type="number" id="idade">
<span id="erroIdade" style="color:red"></span><br>

<button id="validar">Validar</button>
<div id="resultadoValidacao"></div>

✅ Resolução (código completo):

let nome = document.getElementById("nome");
let email = document.getElementById("email");
let idade = document.getElementById("idade");
let btnValidar = document.getElementById("validar");
let resultadoDiv = document.getElementById("resultadoValidacao");

let erroNome = document.getElementById("erroNome");
let erroEmail = document.getElementById("erroEmail");
let erroIdade = document.getElementById("erroIdade");

btnValidar.onclick = function() {
    // Limpar mensagens anteriores
    erroNome.innerHTML = "";
    erroEmail.innerHTML = "";
    erroIdade.innerHTML = "";
    resultadoDiv.innerHTML = "";
    let valido = true;
    
    // Validar nome
    if (nome.value.length < 3) {
        erroNome.innerHTML = "Nome deve ter pelo menos 3 caracteres";
        valido = false;
    }
    
    // Validar e-mail
    if (email.value.indexOf("@") === -1) {
        erroEmail.innerHTML = "E-mail deve conter @";
        valido = false;
    }
    
    // Validar idade
    let idadeNum = Number(idade.value);
    if (idade.value === "" || idadeNum < 1 || idadeNum > 120) {
        erroIdade.innerHTML = "Idade deve ser entre 1 e 120";
        valido = false;
    }
    
    // Mensagem final
    if (valido) {
        resultadoDiv.innerHTML = "✅ Cadastro válido!";
        resultadoDiv.style.color = "green";
    } else {
        resultadoDiv.innerHTML = "❌ Corrija os erros acima";
        resultadoDiv.style.color = "red";
    }
};
